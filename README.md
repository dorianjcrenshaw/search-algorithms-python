# Search Algorithms in Python

This project demonstrates the implementation of fundamental search algorithms used in computer science and artificial intelligence.

## Algorithms Included
- A* Search
- Breadth-First Search (BFS)
- Depth-First Search (DFS)

## Purpose
The goal of this project is to explore how different search strategies perform when navigating a graph structure. These algorithms are commonly used in pathfinding, networking, and AI systems.

## How to Run
1. Clone the repository
2. Run the Python file:
   python search_algorithms.py

## Output
The program prints:
- The shortest path found using A*
- Traversal order for BFS and DFS
